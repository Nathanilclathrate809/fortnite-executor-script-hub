## What It Does

Fortnite XP Executor Hub is a browser-first script tool for Fortnite XP automation workflows. Instead of shipping as a standalone desktop application, it is meant to operate through web interfaces, which makes it usable from a PC browser or, where the interface is supported, a mobile browser.

At its core, the project is focused on repeatable gameplay sequences for XP collection and session handling. It supports chained actions, adjustable run time, status reporting, and refresh behavior designed to help keep the browser session steady during use.

## Key Capabilities

- Automates gameplay routines intended for XP collection

- Supports HTML-based web interface usage
- Provides configurable action chains for different sessions
- Lets you define session duration for shorter or longer runs

- Displays on-screen status indicators while running
- Uses refresh behavior aimed at connection stability
